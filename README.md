# Bob

Bob is a build manager for managing a range of build tasks, including C/C++ compilation and verilation of Verilog/SystemVerilog code.

## Installation

This is a Python utility. Simply clone the project and run with [uv](https://docs.astral.sh/uv/).

```bash
uv run main_consolidated.py
```

## Features
- Flexible framework for adding new build tasks
- Dependency solver
- Incremental and parallel builds
- Expressive input/output references
- Build tool management
